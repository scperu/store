Protector de HTML

Sube un archivo .html, genera una versión protegida y descárgala. Esta versión prioriza mantener la funcionalidad original del HTML.

Selecciona un archivo para empezar.
Vista previa del HTML generado
Esta protección está pensada para usuario promedio. No es seguridad absoluta. No se usa iframe ni sandbox para evitar romper scripts, formularios, enlaces, popups o funciones del HTML original.
Consejo: si quieres más ocultación visual, después puedes ofuscar el JS del archivo generado.
