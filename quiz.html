Si no lee el PDF… (solución)

En algunos Android/Chrome, abrir HTML como file:// puede bloquear recursos externos (CDN).

Solución: abre el HTML desde un servidor local (sin instalar nada): 

python -m http.server 8000

Luego entra a: http://localhost:8000
